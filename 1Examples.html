Below are examples of Section 508 compliant and non-compliant alt text and color contrast specifications.

Alt text 508 Compliant

Let's start at Google's home page. The Hyper Text Markup Language 5 (HTML5) code with alt text for screen readers looks like this:

     <p>Let's start at<a href="http://www.google.com" alt="Google's home page." >Google's home page. </a></p>

Not Alt Text 508 Compliant

Let's start at Google's home page. The non-compliant (Hyper Text Markup Language 5) HTML5 code (omitting alt text for screen readers) looks like this:

     <p> Let's start at <a href="http://www.google.com">Google's home page. </a></p>
